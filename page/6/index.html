Before you read : Indian languages use complex script unlike English letters. Each English letter is composed of a single Unicode letter. But each letter of Indian languages may be composed of more than one Unicode letters. The decomposition is as follows: LetterCode + modifyingSymbolCode Eg: ക്രൊ = ( ക + ്ര + ൊ ) So as you see the code that comes first(ക) need not be the first letter that the user writes(െ).
